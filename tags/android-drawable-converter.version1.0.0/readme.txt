Make sure you have ImageMagick installed: http://www.imagemagick.org/script/binary-releases.php

1) Fill out the settings.ini
2) Then execute java -jar AndroidDrawables.jar

Options
* you can use all the parameters in settings.ini as arguments when executing AndroidDrawables.jar. Ex: "java -jar AndroidDrawables.jar convert_to ldpi"
* you can supply multiple convert_to destinations like "convert_to=ldpi,mdpi,hdpi"
